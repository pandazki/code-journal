# experiments/ — frozen research record

> ⚠️ **Historical.** This tree is the exploratory record behind the observation
> lens — phases, hypotheses, and re-validation rounds, kept as the cited evidence
> base for the shipped feature. It is **not** day-to-day documentation and is not
> maintained. For how the lens actually works today, start from the canonical
> guide [`docs/observation-lens.md`](../docs/observation-lens.md) (中文:
> [`docs/observation-lens.zh.md`](../docs/observation-lens.zh.md)). The living
> confidence ledger is [`docs/observation-lens-record.md`](../docs/observation-lens-record.md).
